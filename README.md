# MyThesis MVP

Minimal desktop-only prototype for thesis topic proposals, matches, messaging, documents, and evaluations.

## Run locally

```bash
npm install
npm run dev
```

## Presentation guide (quick demo flow)

1. Start on **Topics** and use the filters + search to show discovery.
2. Click **Propose topic** and create a new topic as the active user.
3. Switch **Active user** to an eligible role and accept the new topic.
4. Go to **Matches** and open the new workspace.
5. Send a message, add a document (metadata only), and update the evaluation if you are faculty.
6. Use **Reset demo data** to show persistence and the reset capability.
